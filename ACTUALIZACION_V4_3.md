# Actualización a MasterWeb Pro 4.3

1. Reemplaza en GitHub todo el contenido del proyecto anterior por los archivos de esta carpeta.
2. Conserva el mismo Google Apps Script de la versión 4.2; no requiere cambios.
3. Verifica que también se haya subido `assets/plantilla-masterweb.xlsx`.
4. Espera la publicación de GitHub Pages y actualiza con `Ctrl + F5`.

## Prueba recomendada

- Prueba primero con los archivos sueltos habituales.
- Después crea o utiliza un ZIP con los mismos archivos y confirma que los indicadores de cobertura se marquen igual.
- También puedes cargar una parte por ZIP y completar los faltantes como archivos sueltos.
